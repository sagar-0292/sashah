Telegram Setup Guide
Step 1 — Create your bot:
Open Telegram → search @BotFather → send /newbot → follow instructions → copy the token it gives you
Step 2 — Get your Chat ID:
Search @userinfobot in Telegram → send any message → it replies with your ID number
Step 3 — Start your bot:
Search for your new bot by its username → tap Start (important — bot can't message you until you start it)
Step 4 — Paste both values into the fields above and tap Test Message
